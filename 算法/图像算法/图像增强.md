	- 灰度：灰度化是将图像信息，利用图像的亮度值表示出来，从纯白到纯黑(`cv2.cvtColor(img, cv2.COLOR_BGR2GRAY`)
	- 点运算：改变像素的亮度值来调整图像的对比度、亮度或分布特性
		- 线性灰度级变换：一种简单的像素映射方式，通过一个线性公式调整像素值![[Pasted image 20250413185349.png]]
		- 例子
		```
		import cv2
		import numpy as np
		
		# 读取图像
		img = cv2.imread('image.jpg', 0)
		
		# 定义线性变换参数
		a = 1.5  # 对比度增益
		b = 30   # 亮度偏移
		
		# 应用线性灰度级变换
		output_img = np.clip(a * img + b, 0, 255).astype(np.uint8)
		
		# 显示结果
		cv2.imshow('Original Image', img)
		cv2.imshow('Linear Transformed Image', output_img)
		cv2.waitKey(0)
		cv2.destroyAllWindows()
		```
		- 非线性灰度级变换：使用非线性函数（如对数、幂律变换）对像素值进行映射![[Pasted image 20250413195830.png]]
			对数变换
			```
			import cv2
			import numpy as np
			
			# 读取图像
			img = cv2.imread('image.jpg', 0)
			
			# 归一化到 [0, 1] 范围
			normalized_img = img / 255.0
			
			# 应用对数变换
			c = 255 / np.log(1 + np.max(normalized_img))
			log_transformed_img = c * np.log(1 + normalized_img)
			
			# 转换回 [0, 255] 范围
			log_transformed_img = log_transformed_img.astype(np.uint8)
			
			# 显示结果
			cv2.imshow('Original Image', img)
			cv2.imshow('Log Transformed Image', log_transformed_img)
			cv2.waitKey(0)
			cv2.destroyAllWindows()
			```
			gamma矫正
			```
			# 读取图像
			img = cv2.imread('image.jpg', 0)
			
			# 归一化到 [0, 1] 范围
			normalized_img = img / 255.0
			
			# 定义Gamma参数
			gamma = 0.5  # Gamma < 1 增强暗部细节
			
			# 应用Gamma校正
			gamma_corrected_img = np.power(normalized_img, gamma)
			
			# 转换回 [0, 255] 范围
			gamma_corrected_img = (gamma_corrected_img * 255).astype(np.uint8)
			
			# 显示结果
			cv2.imshow('Original Image', img)
			cv2.imshow('Gamma Corrected Image', gamma_corrected_img)
			cv2.waitKey(0)
			cv2.destroyAllWindows()
			```

		- 总结：
			- 如果需要简单调整亮度和对比度，可以使用线性灰度级变换。
			- 如果需要增强暗部细节，可以使用对数变换或者Gamma校正。

	- 领域算法：
		- 平滑：
		
		|滤波方法|原理|使用场景|代码示例|
		|---|---|---|---|
		|**均值滤波**|使用一个滑动窗口（核），计算窗口内所有像素的平均值作为输出像素值。|去除均匀噪声（如高斯白噪声）；平滑图像，但会模糊边缘。|`cv2.blur(img, (5, 5))`![[Pasted image 20250415163428.png]]|
		|**高斯滤波**|类似于均值滤波，但使用的是加权平均，权重由二维高斯分布决定，中心像素权重更高。|去除高斯噪声；平滑图像的同时尽量保留边缘信息|`cv2.GaussianBlur(img, (5, 5), 0)`中心是最高权重![[Pasted image 20250415163518.png]]|
		|**双边滤波**|同时考虑空间邻近性和像素值相似性，既平滑图像又保护边缘。|在去除噪声的同时保留边缘细节，适用于需要高质量边缘保护的场景（如医学图像、人脸美化）。|`cv2.bilateralFilter(img, d=9, sigmaColor=75, sigmaSpace=75)`没有固定滤波核，权重是动态计算的，依赖于每个像素周围的局部信息。|
		|**中值滤波**|使用一个滑动窗口（核），将窗口内所有像素值排序后取中值作为输出像素值。|去除椒盐噪声（Salt-and-Pepper Noise）或孤立点噪声；对保留边缘效果较好，但不适合高密度噪声。|`cv2.medianBlur(img, 5)`|
		 - 锐化：
	
		|算子|原理|使用场景|调用函数示例|
		|---|---|---|---|
		|**Roberts算子**|利用交叉差分求像素值，检测边缘。通过两个卷积核分别计算对角线方向上的差分来检测边缘。|适合用于简单图像中的边缘检测，但对噪声敏感。|![[Pasted image 20250417154930.png]]abs(im[i,j]-im[i+1,j+1])+abs(im[i+1,j]-im[i,j+1])|
		|**Sobel算子**|计算每个像素点的梯度大小和方向，通过在x和y方向上分别应用不同的卷积核进行加权差分。|边缘检测，尤其是对于清晰边缘的提取；适用于各种类型的图像。|![[Pasted image 20250417160909.png]]![[Pasted image 20250417155902.png]]`cv2.Sobel(img, cv2.CV_64F, 1, 0)` 或者 `cv2.Sobel(img, cv2.CV_64F, 0, 1)` 分别对应x和y方向|
		|**Prewitt算子**|类似于Sobel算子，但是使用的卷积核不同，强调的是中心像素周围的平均变化率，没有sobel算子优秀|边缘检测，对于噪声有一定的抑制作用。可能造成高频信息丢失,造成图像模糊|![[Pasted image 20250417155934.png]]![[Pasted image 20250417160536.png]]|
		|**Laplacian算子**|计算图像中每个像素的二阶导数，以突出显示图像中的灰度急剧变化区域，即边缘。|用于强化图像中的细节信息，特别是边缘检测，但容易受到噪声影响。|![[Pasted image 20250417161939.png]]`cv2.Laplacian(img, cv2.CV_64F)`|
		|**LoG算子（拉普拉斯+高斯）**|先使用高斯滤波器平滑图像，然后使用拉普拉斯算子检测边缘，减少噪声的影响。|对于去除噪声同时保留边缘信息非常有用，适用于复杂环境下的边缘检测。|cv2.Laplacian(cv2.GaussianBlur(img, (3, 3), 0), cv2.CV_64F)|
		|**Canny算子**|多阶段算法，包括高斯降噪、sobel计算梯度强度和方向、非极大值抑制(比较附近，取最大的像素)、双阈值处理，检查弱边缘像素，若与强边缘相连，则保留。|提供了更为精确和连续的边缘检测结果，广泛应用于各种边缘检测需求。|![[Pasted image 20250417164001.png]]`cv2.Canny(img, threshold1, threshold2)`|
- 频域算法：先进行傅里叶变换空域转频域，增强完后再逆傅里叶变换![[Pasted image 20250418104533.png]]
	- 频域：频率组成的信息，图像被分解为不同频率的正弦波和余弦波的叠加，这些频率分量描述了图像的周期性特征。对于平滑，频域展示在频域图中间，对于边缘信息，展示在频域图的外围，亮度高低表示频率强度![[Pasted image 20250418095321.png]]
	- 低通滤波：衰减高频信息，使低频信息通过的滤波，作用平滑去噪
	- 高通滤波：显示高频信息，增强高频，作用增强边缘信息
	- 小波变换：对高频进行置零或衰减，实现低频滤波效果；对低频数据置零或衰减，实现高通滤波效果，通过小波基函数找到小波系数，根据目的对小波系数操作
		- 小波基函数：
			- **Haar**: 最简单的正交小波，计算效率高，但不具备连续导数，适合初步分析。
			- **Daubechies (dbN)**: 具有紧支撑性和正交性，db4是最常用的版本之一，适合大多数应用。
			- **Symlets (symN)**: 类似于Daubechies小波，但更加对称，适合要求较高的平滑度的应用。
			- **Coiflets (coifN)**: 具有较高的消失矩和平移不变性，适合需要精确重建的应用。
			- **Biorthogonal (biorNr.Nd)**: 非正交小波，但提供更好的重建效果，特别是对于图像压缩非常有用。
			- **Morlet**: 复小波，主要用于时频分析，适合特征提取等复杂任务。
		- 小波系数：
			- **细节系数（Detail Coefficients）**: 包含高频信息，如边缘、噪声等。对于边缘检测和去噪任务尤为重要。
			- **近似系数（Approximation Coefficients）**: 包含低频信息，代表信号的整体趋势或结构。对于数据压缩和特征提取任务至关重要。
			- **阈值处理**: 去噪过程中常用的技术，通过对细节系数应用阈值来去除噪声，同时保留重要特征。
			- **量化与丢弃**: 在数据压缩中，通过量化减少系数的精度，并丢弃那些对最终结果影响较小的系数，以实现压缩。
	- 照度-反射图像增强:自然景物图像f(x,y)可以表示为光源照度场i和场景中物体反射光的反射场r的乘积，称为图像的照度-反射模型：f=i*r。
	```
	import cv2
	import numpy as np
	import matplotlib.pyplot as plt
	
	# 加载图像并转换为浮点数类型
	image = cv2.imread('example_image.jpg', cv2.IMREAD_GRAYSCALE).astype(np.float32)
	
	# 对数变换
	epsilon = 1e-5  # 防止取对数时出现数值不稳定
	log_image = np.log(image + epsilon)
	
	# 高斯模糊估计照度分量
	sigma = 50  # 控制模糊程度
	illumination = cv2.GaussianBlur(log_image, ksize=(0, 0), sigmaX=sigma)
	
	# 计算反射分量
	reflection = log_image - illumination
	
	# 指数变换恢复反射分量到原始尺度
	enhanced_reflection = np.exp(reflection)
	
	# 显示结果
	plt.figure(figsize=(12, 8))
	
	plt.subplot(221), plt.imshow(image, cmap='gray'), plt.title("Original Image")
	plt.subplot(222), plt.imshow(log_image, cmap='gray'), plt.title("Log Transformed")
	plt.subplot(223), plt.imshow(illumination, cmap='gray'), plt.title("Estimated Illumination")
	plt.subplot(224), plt.imshow(enhanced_reflection, cmap='gray'), plt.title("Enhanced Reflection")
	
	plt.tight_layout()
	plt.show()	
	```

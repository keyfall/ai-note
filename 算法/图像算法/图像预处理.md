#### 基本变换：
- 空域变换： 平移、镜像、旋转、缩放、错切、转置等；其原理是确定变换前与变换后图像中点与点之间的映射关系，明确映射关系后，变换在具体像素点处理中，即为像素点的矩阵变换。在变换过程中，可能因为原图或者处理后的图像存在像素点增多、减少等现象，导致图像的视觉效果变差，因此要运用[[插值运算]]对图像点进行赋值。
- 频域操作：对于图像点像素矩阵庞大的，可以转化为频域，在频域内进行操作，转换方式使用傅里叶变换(`numpy.fft.fft2` 或 `cv2.dft`)，余弦变换(`scipy.fftpack.dct`或`cv2.dct`)
	- 频域操作列表

	|**操作名称**|**简略介绍**|**简单原理**|**常用函数（Python）**|
	|---|---|---|---|
	|**傅里叶变换 (FFT)**|将图像从空间域转换到频域，便于进行频域处理。|通过傅里叶变换将图像表示为不同频率分量的叠加。|`numpy.fft.fft2`, `cv2.dft`|
	|**逆傅里叶变换 (IFFT)**|将经过频域处理后的图像转换回空间域。|通过逆傅里叶变换恢复原始图像或处理后的图像。|`numpy.fft.ifft2`, `cv2.idft`|
	|**低通滤波**|去除高频分量，保留低频分量，用于平滑图像，减少噪声。|构造一个低通滤波器（如理想低通滤波器、高斯低通滤波器），去除高频成分。|自定义滤波器 + `cv2.filter2D` 或直接在频域操作|
	|**高通滤波**|去除低频分量，保留高频分量，用于增强边缘和细节。|构造一个高通滤波器（如理想高通滤波器、拉普拉斯高通滤波器），去除低频成分。|自定义滤波器 + `cv2.filter2D` 或直接在频域操作|
	|**带通/带阻滤波**|选择性地保留或抑制特定频率范围内的分量。|构造带通或带阻滤波器，仅允许或抑制特定频率范围内的分量。|自定义滤波器 + 直接在频域操作|
	|**离散余弦变换 (DCT)**|将图像从空间域转换到频域，常用于图像压缩（如JPEG）。|通过余弦变换将图像表示为一系列余弦波的叠加。|`scipy.fftpack.dct`, `cv2.dct`|
	|**逆离散余弦变换 (IDCT)**|将经过频域处理后的图像转换回空间域。|通过逆余弦变换恢复原始图像或处理后的图像。|`scipy.fftpack.idct`, `cv2.idct`|

	- 去噪：高频分量通常对应于噪声和细节，低频分量对应于平滑区域。低通滤波器通过抑制高频分量，保留低频分量，从而达到去噪的效果，使用低通滤波器或者高斯滤波器,双边滤波器(平滑图像，保持边缘不被去除)
	```
	import cv2
	import numpy as np
	
	def low_pass_filter(shape, radius):
	    rows, cols = shape
	    crow, ccol = rows // 2, cols // 2
	    mask = np.zeros((rows, cols, 2), np.float32)
	    cv2.circle(mask, (ccol, crow), radius, (1, 1), -1)
	    return mask
	
	# 示例：傅里叶变换 + 低通滤波
	img = cv2.imread('image.jpg', 0)
	dft = cv2.dft(np.float32(img), flags=cv2.DFT_COMPLEX_OUTPUT)
	dft_shift = np.fft.fftshift(dft)
	
	mask = low_pass_filter(img.shape, radius=30)
	filtered_dft = dft_shift * mask
	
	idft_shift = np.fft.ifftshift(filtered_dft)
	img_back = cv2.idft(idft_shift)
	img_back = cv2.magnitude(img_back[:, :, 0], img_back[:, :, 1])
	
	cv2.imshow('Denoised Image', img_back)
	cv2.waitKey(0)

	import cv2
	```

	- 增强边缘：高频分量对应于图像中的边缘和细节。高通滤波器通过抑制低频分量，保留高频分量，可以增强图像的边缘和细节。使用拉普拉斯滤波器或者高通滤波器

	```
	def high_pass_filter(shape, radius):
	    rows, cols = shape
	    crow, ccol = rows // 2, cols // 2
	    mask = np.ones((rows, cols, 2), np.float32)
	    cv2.circle(mask, (ccol, crow), radius, (0, 0), -1)
	    return mask
	
	# 示例：傅里叶变换 + 高通滤波
	img = cv2.imread('image.jpg', 0)
	dft = cv2.dft(np.float32(img), flags=cv2.DFT_COMPLEX_OUTPUT)
	dft_shift = np.fft.fftshift(dft)
	
	mask = high_pass_filter(img.shape, radius=30)
	filtered_dft = dft_shift * mask
	
	idft_shift = np.fft.ifftshift(filtered_dft)
	img_back = cv2.idft(idft_shift)
	img_back = cv2.magnitude(img_back[:, :, 0], img_back[:, :, 1])
	
	cv2.imshow('Edge Enhanced Image', img_back)
	cv2.waitKey(0)
	```

	- 去除周期性噪声：期性噪声在频域中表现为特定频率的峰值。带阻滤波器可以抑制这些特定频率范围内的分量，从而去除噪声。
	```
	def band_reject_filter(shape, inner_radius, outer_radius):
	    rows, cols = shape
	    crow, ccol = rows // 2, cols // 2
	    mask = np.ones((rows, cols, 2), np.float32)
	    for i in range(rows):
	        for j in range(cols):
	            dist = np.sqrt((i-crow)**2 + (j-ccol)**2)
	            if inner_radius <= dist <= outer_radius:
	                mask[i, j] = 0
	    return mask
	
	# 示例：傅里叶变换 + 带阻滤波
	img = cv2.imread('image.jpg', 0)
	dft = cv2.dft(np.float32(img), flags=cv2.DFT_COMPLEX_OUTPUT)
	dft_shift = np.fft.fftshift(dft)
	
	mask = band_reject_filter(img.shape, inner_radius=20, outer_radius=50)
	filtered_dft = dft_shift * mask
	
	idft_shift = np.fft.ifftshift(filtered_dft)
	img_back = cv2.idft(idft_shift)
	img_back = cv2.magnitude(img_back[:, :, 0], img_back[:, :, 1])
	
	cv2.imshow('Noise Removed Image', img_back)
	cv2.waitKey(0)
	```
